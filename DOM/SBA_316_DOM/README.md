# DOM To-Do List Application

This project is a simple, dynamic **To-Do List Web App** built for the SBA 316 DOM assessment. It uses **JavaScript DOM and BOM manipulation techniques** to create, manage, and display user tasks with priorities.

---

## Features

- Add tasks with **priority levels** (Low, Medium, High)
- Task **completion** with a "Done" button
- Task **deletion**
- **Clear all** tasks
- **Color-coded** priorities (red, orange, green)
- **Task counter** updates dynamically
- Basic **form validation** using both JavaScript and HTML
- Fully implemented using:
  - DOM manipulation (`createElement`, `appendChild`, `setAttribute`, etc.)
  - BOM methods (`beforeunload`, `navigator.userAgent`)
  - Event-driven programming (`addEventListener`)

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
